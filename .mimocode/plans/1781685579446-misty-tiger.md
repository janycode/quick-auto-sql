# Quick Auto SQL — 付费策略方案

## 一、商业模式定位

**核心模式：SaaS 为主 + 私有化部署授权**

- SaaS 版：用户注册即用，平台托管
- 私有化版：企业购买部署包，按年/永久授权
- 个人/小团队为主，兼顾中小企业

## 四、付费模式设计

### 4.1 推荐方案：订阅制 + 加油包

**订阅制（主力）**：
- 月付：灵活，适合尝试
- 年付：8.5 折优惠，提高留存
- 年付用户赠送 2 个月额度

**加油包（补充）**：
- 基础加油包：¥5 = 100 次 AI 生成（平台提供）
- 高级加油包：¥25 = 600 次 AI 生成（平台提供）
- 加油包永不过期，用完再买

### 4.2 私有化部署（企业版）

| 授权方式 | 价格区间 | 说明 |
|---------|---------|------|
| 按年授权 | ¥2,500 - ¥10,000/年 | 按并发用户数定价 |
| 永久授权 | ¥7,500 - ¥30,000 | 含 1 年技术支持 |
| 定制开发 | 按需报价 | 功能定制、集成 |

私有化部署定价参考：
- 5 人以下团队：¥2,500/年 或 ¥7,500 永久
- 5-20 人团队：¥6,000/年 或 ¥18,000 永久
- 20 人以上：¥10,000/年 或 ¥30,000 永久

### 5.3 配额检查中间件

```typescript
// 每次 AI 调用前检查配额
function requireQuota(type: 'ai_generate' | 'ai_optimize' | 'query_execute') {
  return (req, res, next) => {
    const user = req.auth.user;
    if (!hasQuota(user, type)) {
      return res.status(403).json({
        code: 403,
        message: '今日次数已用完，请升级套餐或购买加油包',
        data: { upgradeUrl: '/pricing', type }
      });
    }
    next();
  };
}
```

## 七、增长策略

### 7.1 获客策略
- **免费版引流**：30 次/月足够体验核心功能
- **邀请裂变**：邀请 1 人 = 双方各得 50 次
- **内容营销**：发布 SQL 优化案例、AI 生成技巧
- **社区运营**：GitHub 开源核心功能，高级功能付费

### 7.2 留存策略
- **签到系统**：培养每日使用习惯
- **积分体系**：消费越多，积分越多
- **升级路径**：免费 → 基础 → 专业，自然转化
- **数据价值**：查询历史、分析报告等数据沉淀

### 7.3 定价心理学
- **锚定效应**：专业版价格作为锚点，基础版显得划算
- **损失厌恶**：免费用户看到"本月剩余 5 次"比"已用 25 次"更有紧迫感
- **社会认同**：展示"已有 XXX 人升级基础版"

## 十、收入预估（保守）

假设：
- 月活用户 1000 人
- 付费转化率 5%（50 人）
- 基础版均价 ¥15/月，专业版均价 ¥39/月
- 基础:专业比例 = 7:3

**月收入预估**：
- 基础版：35 人 × ¥15 = ¥525
- 专业版：15 人 × ¥39 = ¥585
- 加油包：¥250/月（估算）
- **月收入合计：约 ¥1,360**

达到 5000 月收入需要约 3700 月活用户（5% 转化率）。
